# 文档索引

本目录基于产品方案 PDF 整理了后续项目开发的参考文档。

## 文档列表

- `sop.md`：一期业务标准作业流程，覆盖项目创建、素材整理、主题选择、分镜生成、节奏设计、发布方案和导出规则。
- `prd.md`：一期产品需求文档，覆盖产品定位、目标用户、功能范围、页面需求、数据模型、里程碑和风险控制。
- `api.md`：一期 API 设计文档，覆盖接口规范、核心数据结构、生成型接口、导出接口和 JSON Schema 建议。

## 使用建议

- 产品和运营优先阅读 `prd.md` 与 `sop.md`。
- 前后端联调优先阅读 `api.md`。
- 后续进入二期自动粗剪时，优先复用 `api.md` 中的分镜 JSON 结构。
